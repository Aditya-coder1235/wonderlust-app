#  Wonderlust

**Wonderlust** is a full-stack travel application built with the **MERN Stack (MongoDB, Express, React, Node.js)**.  
It allows users to explore, create, and review beautiful travel destinations around the world.  
The goal of Wonderlust is to inspire travelers to discover new places and share their adventures with others.

---

##  Features

-  **Explore Destinations:** Browse a collection of stunning places around the world.
-  **User Authentication:** Secure login and signup with Passport.js.
-  **Reviews & Ratings:** Add, edit, and view reviews on different destinations.
-  **Image Uploads:** Upload destination images using Cloudinary.
